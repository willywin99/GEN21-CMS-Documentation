Used to Parameter in Program Budget Definition

 Open the Budget Parameter   

1.    Login to Gen21CMS

2.    Open the Application Configuration (Accounts ->  Budget Parameters).     

 

 

Steps Adding New Budget Parameters

1. Open Budget Parameters 

2. Create new Budget Parameters,

      a. Right click on mouse button on the header field and choose

          “add row” to create new Budget Parameters

      b. Define the Param Code, Description, Currency Code and Value

      c. Right click on mouse button and choose “save”

Steps Editing Budget Parameters

1. Open Budget Parameters

2. Edit specific Budget Parameters ,

      a. Right click on mouse button and choose “edit mode”.

      b. Edit the Description, Currency Code and Value

      c. Right click on mouse button and choose “save”

Steps Deleting Budget Parameters

1. Open Budget Parameters

2. Delete Budget Parameters,

      a. Right click on mouse button and choose “edit mode”

      b. Right click on mouse button and choose “delete”
