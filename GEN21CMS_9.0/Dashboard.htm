Dashboard

Data Visualization of Content Management System application. We can simply see it as charts.

 

How to Add Charts

  1. Open DashBoard menu. (File -> DashBoard)
  2. Click Chart Property button.
  3. In the Chart Property pop-up, click Insert Record button. (Marked as plus/+ icon)
  4. In the Select Chart Type pop-up, select the chart type adding to dashboard.
  5. Click OK button.
