Used to reduce episodes for selected program.

Steps to proceed

  1. Click on the Program Management > Purchase Contract > Program Info > Reduce Episode menu item.
  2. To reduce episode, check on the check box and select OK.

 Field Name

 Type

 Description

Constraints

 Reduce Checkbox  Reduce  Mandatory 
 Epi # Number Episode number of the program  Mandatory, By System 
 Episode Title Text  By default this is the name of the Program added to the end, where 'n' is the epi #.  Mandatory, By System 
 Epi No Number  Number of Episode  Mandatory, By System 
 Suffix Text  Suffix  Mandatory, By System
