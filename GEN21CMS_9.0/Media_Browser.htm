Media Browser

 

Description

This indicates that this media has been booked and cannot be used by other borrowers, be it blank media or non-blank / Programs that already have material.

 

Application Configuration

Module function settings are based on configuration according to user requirements

Step

   Open Media Browser

   1.  Loggin to Gen21 CMS
   2.  Open Module Media Browser  ( Media Library --> Media Browser )

 

Searching

Searching is a facility for searching data that has been created, and the search is tailored to  the needs of the user based on existing criteria, for example Media Category, Media Type, Media Format & Content Name.

                     

   Mark For Cek Out Media Browser

     1.  Press klik pop up Requestor Name

     2.  Press button Ok

     3.  Klik Pop Up menu Borrower & Request Date

     4.  Press save from main menu

                              5.  Drag & Drop from grid to detail check out.
                              6.  Press Save from main menu

    Un-Mark For Cek Out Media

     1.  Cek Box & select media cek out in detail UnMark

     2.  Drag & drop media

     3.  Press save from main menu
