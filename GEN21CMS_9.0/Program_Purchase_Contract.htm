Used to create and maintain information on Purchase Contract transactions. Purchase transactions are defined as those conducted between the Distributor / Broadcaster (who acquires the material) as the buyer and external parties (who typically are actual content producers eg. Production Houses, Movie Studios etc.) as the seller(s).

Steps Program Purchase Contract

1. Login to Gen21CMS

2. Open the Program Purchase Contract (Menu-> Program Maanagement-> program Purchase Contract).

3. program Purchase Contract window Contains 2 main Tab windows.

4. Search Tab ( Default Tab upon open the Program purchase contract) -> Used to Search The Existing Purchase contracts.

Purchase contract window - Form Tab is divided into 2 logical parts:

The "Contract" - is also called "PO Header", stores financial and administrative information of a PO. Description starts in contract section.

The "Program & Episode" - contains detailed material contents of a PO defined as Programs and Episodes.

Lookup "Program Info" section for moore details.
