Used to change/maintain their login passwords. It is good practice to change one's password used to access system from time to time for the sake of security.

Open the Change Password

  1. Login to Gen21 CMS.
  2. Open the Change Password (Master > Change Password).

Steps for Change Password

  1. Open the Gen21 CMS.
  2. Open the Change Password window.
  3. Enter old password.
  4. Enter new password.
  5. Enter New Password as Repeat.
  6. Click OK.

Field Definitions

 Field Name

 Description

 Brief Description

 Constraints

 Old Password    User Password (currently used for login to Application).  Mandatory
 New Password    The new User Password that you have chosen for log in to the application.  Mandatory
 Confirm Password    A confirmation field used to enter the new password for the second time. This is to ensure the password is meant to be exactly the way it was first typed.  Mandatory
