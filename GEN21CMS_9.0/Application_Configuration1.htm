Used for maintaining the application configuration. Configuration can be set for general or basic needs also for specific category.

Basic Setup for Application Configuration

Got Access Admin rights for EDP menu to GEN21CMS.

Open the Application Configuration window

  1. Login to Gen21 CMS
  2. Open the Application Configuration (Master > Application Configuration)

Application configuration window contains 3 sections Application configuration

  1. List of application configuration based on groups.
  2. Display list of configuration based on selected group.
  3. Details of configuration in list of configurations.

Edit the Configuration Values

  1. Press edit from Top menu bar or Press edit from pop-up menu or press F4.
  2. Select the config code for Edit.
  3. Edit the Description or Edit Parameter value.
  4. Press Save from top menu or press Save from pop-up menu or press CTRL+S.
