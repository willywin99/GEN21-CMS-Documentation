Receive Blank Media

 

Description

Create and maintain all blank tape/media inventory. Typically performed upon receipt of new batches of blank tape/media shipments, the output of this function is a set of tape labels readyfor physical labeling on the tapes.

 

Application Configuration

Module function settings are based on configuration according to user requirements.

 

Step

 

Open Receive Blank Media

   1.  Loggin to Gen21 CMS
   2.  Open Module Receive Blank Media  ( Media Library --> Receive Blank Media )

               
 

Searching

Searching is a facility for searching data that has been created, and the search is customized t the needs of the user based on existing criteria, for example by Media Type, PO#, Arr Date Or Channel.


                                                                                                                                                                                                                                                            

    Add Receive Blank Media Header

     1.  Press Right Button for open the popup menu

     2.  Press edit in popup menu

     3.  Press add in popup menu

     4.  Enter the values in modifiable fields / Insert Data

     5.  Press save from main menu

.

    Delete Receive Blank Media

   1.   Press Right Button from program detail for open the popup menu

   2.   Press mouse right button for open the popup menu

   3.   Select the record in list

   4.   Press Edit in Popup menu

   5.   Press Delete in Popup menu 
   6.   Press Save from Main menu
