Holiday

Description

Holiday information related to the details of the attribute.

Application Configuration

               Module function settings are based on configuration according to user requirements

Steps

                Open Parameter Exchange Rate

1.       Loggin to Gen21 CMS

2.       Open the module Holiday  ( Program Management --> Manage Episode Detail  --> Parameter --> Holiday )

        

                      Add Holiday

                      1. Open the Holiday window
                      2. Press mouse right button for open the popup menu
                      3. Press Edit in Popup menu
                      4. Press Add in Popup menu
                      5. Holiday Code Generated from system
                      6. Enter Holiday Code Description
                      7. Press Save from Main menu

                      Delete Holiday

                     1. Open the Holiday window
                     2. Press mouse right button for open the popup menu
                     3. Select the record in list
                     4. Press Edit in Popup menu
                     5. Press Delete in Popup menu 
                     6. Press Save from Main menu

                If Record is already used by transactions, Deletion can not possible
