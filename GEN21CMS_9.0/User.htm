Used to create and maintain all user information.

Basic Setup for User

Got Access Admin rights for EDP menu to GEN21 CMS.

Open the User

  1. Log in to GEN21 CMS.
  2. Open the User. (Master - User).

User window contains 4 sections

  1. User List = > List of all user.
  2. User Detail = > Display the details of selected user.
  3. User group = > Display the user group details of selected user.
  4. Channel Group = > Display the allocated channel to user.
Steps for Create User

  1. Open the GEN21 CMS
  2. Open the User window
  3. Right click on mouse for Open the pop-up menu.
  4. Click Add.
  5. Enter User code.
  6. Enter Username.
  7. Enter User email.
  8. Enter Non Active Date.
  9. Select the Option for change the password on log on.
  10. Click Save from main menu.

Steps for adding the User Group

  1. Open the GEN21 CMS.
  2. Open the User window.
  3. Search for User.
  4. Right click on User Group section for open the pop-up menu.
  5. Select the user group from list box on column user group.
  6. Click Save from Main menu.

Steps for adding channel access to user

  1. Open the GEN21CMS.
  2. Open the User window.
  3. Search for User.
  4. Right click on Channel area for open the pop-up menu.
  5. Click Save from main menu.
