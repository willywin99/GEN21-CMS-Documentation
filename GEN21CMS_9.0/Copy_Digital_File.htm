Copy Digital File

Purpose

The use of programs that already in receive program and could not return again.

Note : To make program returns Already in Copy Digital File program must create new PO.

 

Steps to Proceed

  1. Click on the Media Library > Copy Digital File menu item.
    The starting Copy Digital File screen is displayed.

    As before, the function start with a 'Search' screen to locate the media to be worked on. Enter any search criteria(s) in available fields in search field and click 'Search' button to commence the search.
  2. The output of the search is a list of all records found by the search
     
  3. Select 1 or more media to be worked on by clicking them individually in the left check-box (or select/deselect ALL by clicking the check-box heading)
  4. Perform specific activities on the selected records by clicking any of the available functions listed as the horizontal menus. Please refer to the following dedicated subsections for details on each chosen activity:
      • Copy Digital File
      • Media Usage
      • Media Properties
      • Historical Activities
