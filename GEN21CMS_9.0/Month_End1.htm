Used to provide facility to create different month end.

Basic Setup for Month End

Access to Master Menu.

Open the Month End

  1. Open the Gen21 CMS.
  2. Open the Month End (Master > Month End).




Steps for Month End

  1. Open Month End window (Master > Month End).
  2. Right click on Month End detail section for open the pop-up menu.
  3. Fill in Month, Year, and Close.
  4. Click Save from the main menu.

 

Field Definitions

 Name

 Type

 Description

 Constraints

 Month  Dropdown  Selected Month  Mandatory
 Year  Number  Selected Year  Mandatory
 Closing Date  Date  The date when user filled this form, default: today  Generated by System, default: today
 Close  Checkbox  Month isClose  Generated by System, default: checked
 Remarks  Text  Remarks of Month End  Optional
 Update User  Text  The user who fill this form  Display Only, Generated by System
 Update Date  Date  The date when this form is filled and updated last  Display Only, Generated by System
 Usage Report PDF  Button  Button to upload report  Optional
