Used to Program Budget Definition in Accounts.

 

Open the Program Budget Definition

1.    Login to Gen21CMS

2.    Open the Application Configuration (Account -> Program Budget).

Browse Tab

 

Steps for add Program Budget Definition      

1.     Open the window Program Budget Definition 

2.     Press Mouse right button for open the popup menu

3.     Press edit in popup menu

4.     Press add in popup menu

5.     Enter Budget Code  

6.     Enter Contract#

7.     Enter Episode Group/Shooting Loc

8.     Enter Production From/To/Air Date From/To/Duration

9.     Enter Sposored By/Amount

10.   Enter Ref By/Ref Date

11.   Enter Remarks

12.   Enter Paramerer/Currency/Value

13.   Press save from main menu

 

Steps for Delete Program Budget Definition  

1.      Open the window Program  Budget Definition

2.      Press Mouse right button for open the popup menu

3.      Select the Record in list

4.      Press Edit in popup menu

5.      Press Delete in popup menu

6.      Press save from main menu
