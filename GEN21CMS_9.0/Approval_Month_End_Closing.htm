Used to Apply Budget on the parameters that have been made in Manage Program Budgets

           Open The Program Approval Month End Closing

1.      Login to Gen21CMS.

2.      Open the Application Configuration (Menu-> Account -> Approval Month End Closing).

    

Steps For Add Approval Month End Closing   

1.      Open the window Approval Month End Closin 

2.      Enter P.O.#  

3.      Enter Program Name

4.      Enter EpisodeName 

5.      Press Retrieve from Approval Month End Closing

6.      Check to select

7.      Press to Approval User 1 or Approval User 2

8.      Press to Process

 

 

 

Field Definition

 

 Name

 Type

 Descriptions

 Constraints

 PO Number  Alphanumeric

 Purchase Order Number

  Mandatory
 Program Name  Text  Name of the Program  Mandatory
 Episode Name  Text  Episode number of this program, this is automatically numbered from 1 to the total no of episodes stated at the PO program info level.  Mandatory
 Retrieve  Button  To search for PO Number, Program Name and Episode Name that has been selected.  Mandatory
 Approval User 1  Button  Process Approval User 1 Month End Closing  Mandatory, Optional
 Approval User 2  Button  Process Approval User 2 Month End Closing  Mandatory, Optional
 Reject  Button  Reject process Approval Month End Closing  Optional
 Process  Button  For process Approval Month End Closing  Mandatory, Optional
 Cancel  Button  Cancel process Approval Month End Closing  Optional
