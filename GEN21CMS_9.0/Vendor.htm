Vendor

Description

                   Creation and maintenance of detailed Vendor definition.

Application Configuration

               Module function settings are based on configuration according to user requirements

 

 

Steps

                Open Parameter Vendor

1.       Loggin to Gen21 CMS

2.       Open the module Vendor  ( Program Management --> Manage Episode Detail  --> Parameter --> Vendor )

      

                  Add Vendor

                  1.  Open Module Parameter Vendor
                  2.  Open the Vendor Window
                  3.  Press Right Button for open the popup  menu
                  4.  Press edit in popup menu
                  5.  Press add in popup menu
                  6.  Code generated from system
                  7.  Enter Name
                  8.  Enter Country
                  9.  Enter City
                10.  Enter Address
                11.  Enter Pos Code
                12.  Enter Phone Number
                13.  Enter Fax
                14.  Enter Email
                15.  Enter Contact Person
                16.  Press Save From Main Menu 

           

                  Delete Vendor

                  1. Open the Vendor window
                  2. Press mouse right button for open the popup menu
                  3. Select the record in list
                  4. Press Edit in Popup menu
                  5. Press Delete in Popup menu
                  6. Press Save from Main menu

                If Record is already used by transactions, Deletion can not possible
