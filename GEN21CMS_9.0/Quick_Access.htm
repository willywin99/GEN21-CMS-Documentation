Quick Access

How to add quick access

  1. Click Add button from the left bottom menu.
  2. Select the option in the list.
  3. Click Ok.

 

How to Edit Quick Access

  1. Click Edit Property button from the left bottom menu.
  2. Select the property and change editable values.
  3. Click OK.

 

How to Delete Quick Access

  1. Click Edit Property button from the left bottom menu.
  2. Select the property and change editable values.
  3. Click Delete Records.
  4. Click OK.
