Multiple Cek In Out

 

        Multple Cek Out Blank Media

Description


Working in synchronization with Receive Blank Media ,this function manages the circulation (borrow & return) of multiple type Blank Media items at once in a single operation by appropriate users in the organization.


 

Application Configuration

Module function settings are based on configuration according to user requirements

 

Step

   Open Multiple Cek Out Blank Media

   1.  Loggin to Gen21 CMS
   2.  Open Module Multiple Cek OuBlank Media  ( Media Library --> Multiple Cek Out Blank
        Media )   
              

    Add Blank Media Cek In Out

     1.  Press Right Button for open the popup menu

     2.  Press edit in popup menu

     3.  Press add in popup menu

     4.  Select Field Mandatory

     5.  Press save from main menu

 

    Searching

     Searching is a facility for searching data that has been created, and the search is tailor to the needs of the user based on existing criteria, for example trans#, Inte Reference, Out Date, Media Tape, Program Code, Borrower Name,
    Borrower Departement.

                          

.                      

  

                       Browse

                   Displays the list of records matching the input search criteria. Simply use th arrow buttons or click on the desired single record on the list, Also, this menu  has “Browse Tab” to display all (nama menu) that be existed in the system. Use can find the specific (nama menu) by filter the name on the filter facility.

Steps on the browser screen :

-  Click on browse tab

-  Find specific (nama menu) filter on the drop down filter list feature

-  Type on the text box filter of (nama menu) that want to search
