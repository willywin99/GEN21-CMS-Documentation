Episode Cost

Purpose

Detail information episode cost.

Steps to Proceed

  1. Click on the Program Management > Manage Episode Details > Episode Details > Episode Cost menu item.
  2. This is information from Program Purchase Contract > PO# > Program Info > Episode and Runs.

 

Field Definition

Episode Cost

 Field Name

 Type

 Brief Description

 License Cost  Decimal  License Cost in home currency
 Adj Cost  Decimal  Additional Cost in home currency
 Org Lic Cost  Decimal  License fees in currency other than Rupiah
 Adj Org Cost  Decimal  Actual cost in currency other than Rupiah
 Actual Cost  Decimal  Actual cost in home currency
 Actual Org Cost  Decimal  Actual cost in currency other than Rupiah
 Exchange Rate  Decimal  Exchange Rate
 Currency  Text  Currency

 

Runs

 Field Name

 Type

 Brief Description

 Run #  Number  Sequential airing/run number of the episode from 1 to n as recorded in episode sub-tab above
 Run Aired  Check-box  System initiated aired episode indicator
 Show Name  Text  The banner name as listed in slot listing of the day
 Air Date  Date  Airing date of the episode
 Air Time  Alphanumeric  Airing time of the episode
 Start Time  Alphanumeric  Exact start time of the last airing
 End Time  Alphanumeric  Exact end time of the last airing
 Amor %  Decimal  The amortized value of the episode on this run
 Exch Rate  Decimal  The rupiah (default currency) exchange rate per foreign currency unit used at the time of the purchase
 Amor Val (Frg)  Decimal  Amortization value scheduled for this run (in foreign currency)
 Amor Val (Rp)  Decimal  Amortization value scheduled for this run (in home currency)
 Amor Taken  Decimal  Amortization value taken for this run
 Balance  Number  The remaining balance, non-modifiable and calculates from the Amount field at the Header section minus the running payments
 Revenue  Decimal  Revenue sharing indicator
 TV Rating  Text  The post-airing actual TV Rating figure for the episode on the show. In free-format textual type (can include any special characters)
 Sharing  Text  Additional condition indicators and percentages owed to third party, in case of more sharing scheme than those defined above (optional)
 Update User  Text  Captured from user Login code (non modifiable). The user who last performed the modification on this PO
 Update Date  Date  Date of the last data updating (non modifiable)
