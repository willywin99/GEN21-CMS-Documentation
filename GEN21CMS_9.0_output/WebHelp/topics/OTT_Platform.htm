OTT Platform

How to add OTT Platform

  1. Open OTT Platform window (Program Management > Parameter Program Management > OTT Platform)
  2. Right clik on header and select Add Row
  3. Fill in Name Field
  4. Right click on header and select Save

How to Update OTT Platform

  1. Open OTT Platform window (Program Management > Parameter Program Management > OTT Platform).
  2. Right click on header and select Edit Mode.
  3. Change the name field of OTT Platform.
  4. Right click on header and click Save.

How to delete OTT Platform

  1. Open OTT Platform window (Program Management > Parameter Program Management > OTT Platform).
  2. Select the OTT Platform which one to be deleted.
  3. Right click on header and select Edit Mode.
  4. Right click on header and select Delete Row.
  5. Click Yes to confirm the deletion.
