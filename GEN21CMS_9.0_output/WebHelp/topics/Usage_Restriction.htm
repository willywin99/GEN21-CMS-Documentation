Usage Restriction

Purpose              

Creation and maintenance of Usage Restriction definition.
This is where all possible 'usage restriction' types related to program material are listed & maintained. eg. defined as time slot or content nature related.

 

Steps to proceed

1.      Click on the Program Management > Manage Episode Details > Parametes > Usage Restriction menu item.
2.      To add new Usage Restriction, right click at the header and select Edit, right click at the header again and select Add, enter the Usage Restriction in modifiable fields then click the save button.
3.      To modify Usage Restriction, select the media from the browse list, right click at the header of Form tab and select Edit, retype over any modifiable fields and click the save button .  

To delete Usage Restriction, select the media from the browse list, right click at the header and select Edit, right click again at the header and select delete. 

4.           Note: Deletions are not possible when the records in question are already used in relation to any other entities in the application.
