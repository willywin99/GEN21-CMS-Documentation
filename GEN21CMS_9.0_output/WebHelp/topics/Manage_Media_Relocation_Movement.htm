Manage Media Relocation

    Purpose

   to move the program to the Receive Program Media and archive on the Receive Stock Shoot

    Steps to proceed

1.      Click on the  Media Library> Manage Media Relocation  menu item. The starting Media Browser screen is displayed

2.    Select the Program or Archive

3.      Select the PO No to if Program

4.      Secelct the Shelf No to if Archive

5.      Press Retrieve 

6.      Select the program for move

7.      And press Relocate
