Predefined Query

GEN21 CMS Application contains some Pre-defined Queries. Those queries can be modified by User with Authorization level.

Basic Setup for Pre-defined Query

Access to Master menu.

Open the Pre-defined Query

  1. Open the GEN21 CMS.
  2. Open the Predefined Query. (Master > Predefined Query)

Steps for Predefined Query

  1. Open the Predefined Query window.
  2. Select the Report in right panel.
  3. Enter the required parameters.
  4. Click Retrieve button.
  5. For Save as Result, click Save As button.
